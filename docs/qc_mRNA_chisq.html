Introduction

This report mirrors qc_rRNA.html but evaluates an alternative site-calling strategy based on a per-site contingency-table (chi-squared) test rather than simple ratio/depth thresholds.

For every uridine (T) position we build a 2x2 table comparing the treated and (pooled) untreated samples:

Conversion (C) No conversion (T)
Treated a b
Untreated c d

and test the null hypothesis that the conversion rate is the same in both samples (\(X^2 = \frac{n(ad-bc)^2}{(a+b)(c+d)(a+c)(b+d)}\), 1 d.f.). A site is called pseudouridylated when:

  1. the chi-squared Benjamini-Hochberg FDR < cutoff, and
  2. the conversion-rate difference (treated − untreated) > cutoff.

We ran this strategy on the same four cell lines: CL18870, CL18909, CL18916, and CL19099 (four treated replicates each).

We compare our results against two published datasets:

  • BACS — PMID 39349603
  • PRAISE — PMID 36997645

Comparing pooled data among cell lines

After per-replicate chi-squared filtering, sites for each cell line are pooled (pooled_sites_chisq/pooled_mRNA.txt): treated and untreated counts are summed across the four replicates and the chi-squared test is re-run on the pooled contingency table, yielding pooled pval, fdr, and rate_diff statistics alongside the pooled %pU ratio. Each row also records n_samples, the number of individual replicates in which the site was detected.

We explore the effect of two filtering dimensions simultaneously, using the same four FDR / rate-difference combinations introduced above together with a minimum-replicate filter:

Dimension Values
FDR / rate-difference combination the four combinations above
Minimum number of replicates (n_samples) 1, 2, 3, 4

Stricter combinations (lower FDR, larger rate difference) reduce noise at the cost of fewer sites; requiring detection in multiple replicates is a strong reproducibility filter.

count_sites_chisq <- function(dt, combo, min_nsamp) {
  sum(dt$fdr < combo$fdr & dt$rate_diff > combo$diff & dt$n_samples >= min_nsamp)
}

build_count_grid_chisq <- function(dt) {
  do.call(rbind, lapply(seq_along(CUTOFF_COMBOS), function(i) {
    combo <- CUTOFF_COMBOS[[i]]
    data.frame(
      combo_label = combo$label,
      min_nsamp   = NSAMP_CUTS_POOLED,
      n_sites     = vapply(NSAMP_CUTS_POOLED,
                           function(n) count_sites_chisq(dt, combo, n),

      combo_label = factor(combo_label, levels = COMBO_LABELS),
      nsamp_label = factor(paste0("n_samples>=", min_nsamp),

    )
}

Cross-cell-line comparison of site counts

We use the Jaccard index to quantify how much the pooled, significance -filtered site sets overlap between every pair of cell lines. For each combination of FDR / rate-difference cutoff and minimum-replicate filter, sites passing both filters in each cell line are extracted and the pairwise Jaccard is computed:

\[J(A, B) = \frac{|A \cap B|}{|A \cup B|}\]

A high Jaccard index means the two cell lines call largely the same pooled mRNA positions. Each cell shows the Jaccard score and, in parentheses, the number of sites in the intersection. Rows = minimum n_samples; columns = FDR / rate-difference combination.

jaccard_pooled_rows <- do.call(rbind, lapply(cl_pairs_v_chisq, function(p) {
  do.call(rbind, lapply(seq_along(CUTOFF_COMBOS), function(i) {
    combo <- CUTOFF_COMBOS[[i]]
    do.call(rbind, lapply(NSAMP_CUTS_POOLED, function(n) {
      dt1 <- dat_list_pooled_chisq[[p[1]]]
      dt2 <- dat_list_pooled_chisq[[p[2]]]
      sites1 <- dt1$site[dt1$fdr < combo$fdr & dt1$rate_diff > combo$diff & dt1$n_samples >= n]
      sites2 <- dt2$site[dt2$fdr < combo$fdr & dt2$rate_diff > combo$diff & dt2$n_samples >= n]
      n_int  <- length(intersect(sites1, sites2))
      n_uni  <- length(union(sites1, sites2))
      j      <- if (n_uni == 0) NA_real_ else n_int / n_uni
      data.frame(cl1 = p[1], cl2 = p[2],
                 combo_idx = i, combo_label = combo$label, min_nsamp = n,
                 jaccard = round(j, 3), n_intersect = n_int)
    }))
  }))
}))

Pearson’s r of %pU between cell lines

To assess whether the same sites are similarly modified across cell lines, we compute Pearson’s r on the pooled %pU values at sites passing the FDR / rate-difference filter and the minimum-replicate filter in both cell lines of each pair. High r across cell lines suggests that the detected modification pattern is consistent — i.e., the same mRNA sites are strongly modified regardless of cell line.

Sites that pass in one cell line but not the other are excluded from the correlation (they contribute to the Jaccard-style overlap question, not the quantitative agreement question).

pearson_pair_chisq <- function(dt1, dt2, combo, min_nsamp) {
  pass1 <- dt1[fdr < combo$fdr & rate_diff > combo$diff & n_samples >= min_nsamp,
               .(site, ratio1 = ratio)]
  pass2 <- dt2[fdr < combo$fdr & rate_diff > combo$diff & n_samples >= min_nsamp,
               .(site, ratio2 = ratio)]
  merged <- merge(pass1, pass2, by = "site")
  if (nrow(merged) < 3) return(data.frame(r = NA_real_, n_common = nrow(merged)))
  data.frame(
    r        = cor(merged$ratio1, merged$ratio2, method = "pearson"),
    n_common = nrow(merged)
  )
}

cor_pooled_grid <- do.call(rbind, lapply(cl_pairs_v_chisq, function(p) {
  do.call(rbind, lapply(seq_along(CUTOFF_COMBOS), function(i) {
    combo <- CUTOFF_COMBOS[[i]]
    do.call(rbind, lapply(NSAMP_CUTS_POOLED, function(n) {
      res <- pearson_pair_chisq(
        dat_list_pooled_chisq[[p[1]]], dat_list_pooled_chisq[[p[2]]],
        combo = combo, min_nsamp = n
      )
      data.frame(cl1 = p[1], cl2 = p[2],
                 combo_idx = i, combo_label = combo$label, min_nsamp = n,
                 pearson_r = round(res$r, 3), n_common = res$n_common)
    }))
  }))
}))

Comparing with published data

We benchmark our pooled chi-squared mRNA calls against two independent published datasets. Overlap with validated sites confirms that the chi-squared site-calling strategy is correctly identifying true pseudouridine modifications in mRNA.

  • BACS (PMID 39349603) detects pU via antibody-based capture; it reports absolute Ψ levels for HeLa polyA+ RNA (Supplementary Table 8).
  • PRAISE (PMID 36997645) detects pU via the deletion signature at reverse- transcription termination; it reports deletion ratios for two replicates in HEK293T mRNAs and ncRNAs (Supplementary Dataset 2 — filtered to mRNA only).

Both published datasets use genomic coordinates, matching our pooled mRNA output (chromosome + genomic position + strand). BACS mRNA sites with feature == "ncRNA" are excluded. PRAISE sites with NR_ or XR_ accession prefixes (ncRNA) are excluded; only NM_ and XM_ (mRNA) entries are kept.

For each cell line and each of the four FDR / rate-difference cutoff combinations introduced above we report: (a) how many of our pooled, significance-filtered sites overlap with BACS sites, (b) how many overlap with PRAISE sites, and (c) scatter plots comparing the quantitative %pU or deletion ratio values at overlapping positions.
