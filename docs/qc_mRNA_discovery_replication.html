JOINT_DIR <- "/project/xinhe/xsun/pumseq/pumseq_processed/joint"
N_SPLITS <- 3
FDR_THRESH <- 0.01
RATEDIFF_THRESH <- 0.05
DEPTH_THRESH <- 20

Introduction

We split the cell lines into a small discovery subset and a larger replication subset, call sites by pooling reads within the discovery subset only (same “joint” pooling + chi-squared approach used throughout this project), then check — in the independent replication subset — (1) what fraction of discovery-called sites are also significant, and (2) how well %pU at those sites correlates between discovery and replication.

Design: 51 cell lines split into discovery (~20%, 10 cell lines) and replication (~80%, 41 cell lines), independently for 3 random splits (different cell lines in the discovery pool each time), to see how much the result depends on which specific cell lines end up in discovery. Site calling in both pools uses the same thresholds as the standard joint pipeline (FDR < 0.01, rate_diff > 0.05, depth >= 20).

load_split <- function(i) {
  disc_dir <- file.path(JOINT_DIR, paste0("discovery_split", i, "_mRNA"))
  repl_dir <- file.path(JOINT_DIR, paste0("replication_split", i, "_mRNA"))

  discovery_sites <- fread(file.path(disc_dir, "joint_mRNA.global_sites.txt.gz"))
  discovery_sites <- discovery_sites[fdr < FDR_THRESH]   # apply FDR cutoff to discovery calls too
  replication_full <- fread(file.path(repl_dir, "joint_mRNA.pseusite.txt.gz"))

  matched <- merge(
    discovery_sites, replication_full,
    by = c("ref", "pos", "strand"), suffixes = c(".disc", ".repl")
  )
  matched[, split := i]
  matched
}

all_splits <- rbindlist(lapply(seq_len(N_SPLITS), load_split))

Replication rate

For each discovery-called site, checking whether it’s also significant under the same thresholds (FDR < 0.01, rate_diff > 0.05, depth >= 20) in the independent replication pool.

all_splits[, replicated := (fdr.repl < FDR_THRESH) & (rate_diff.repl > RATEDIFF_THRESH) & (depth.repl >= DEPTH_THRESH)]
all_splits[, tested_in_repl := depth.repl >= DEPTH_THRESH]

repl_summary <- all_splits[, .(
  n_discovery_sites = .N,
  n_testable_in_replication = sum(tested_in_repl),
  n_replicated = sum(replicated, na.rm = TRUE),
  replication_rate = round(sum(replicated, na.rm = TRUE) / .N, 3)
), by = split]

Column definitions:

  • n_discovery_sites: number of sites called in the discovery pool.
  • n_testable_in_replication: of those, the number that have adequate depth (>= 20) in the replication pool — i.e. sites with enough replication-pool reads to actually be evaluated for significance, whether or not they replicated.
  • n_replicated: of the discovery sites, the number that are also significant in the replication pool (FDR < 0.01, rate_diff > 0.05, depth >= 20).
  • replication_rate: n_replicated / n_discovery_sites.
ggplot(repl_summary, aes(x = factor(split), y = replication_rate)) +

Notes

  • Discovery-pool depth (above) directly addresses the concern that calling sites from only ~10 pooled cell lines may be depth-limited compared to the full 51-cell-line joint pool.
  • Replication rate and %pU correlation are reported per split (not just pooled) to show how much the result depends on which specific cell lines happen to land in the discovery pool.
  • n_testable_in_replication turns out to be nearly identical to n_discovery_sites (see the note under the replication-rate table): the larger replication pool gives essentially every discovery site adequate depth to be tested. So non-replication here reflects sites that genuinely fail to reproduce as significant in independent samples, not sites that were too shallow in replication to evaluate.
