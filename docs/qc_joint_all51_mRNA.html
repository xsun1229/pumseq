Introduction

Quality control of the joint cross-cell-line mRNA pU site calling across all 51 cell lines (with problematic replicates dropped based on BACS r < 0.3 in both reps).

The joint strategy pools treated and untreated reads across all cell lines, runs a single chi-squared test per uridine position, then applies genome-wide BH-FDR. The resulting global site list defines a common feature set.

For each global site, per-cell-line %pU is computed as C/(C+T) from each cell line’s raw mpileup data → ratio matrix (sites × 51 cell lines).

Motif analysis around pU sites

We extract ±10 bp flanking sequences around each global pU site from the reference genome, then compute position-specific nucleotide frequencies. The pU site (position 0) is always T on the sense strand (T→Ψ conversion). For sites on the minus strand, the reverse complement is taken so position 0 is always T.

Known mRNA pseudouridine motifs from the literature:

  • TΨC / GUΨC — the most commonly reported consensus, associated with TRUB1-dependent mRNA pseudouridylation
  • UΨAN — enriched in some studies
  • PUS7 targets UGΨAR

BACS overlap restricted to UTR/CDS sites

BACS sites are exclusively in 5’UTR/CDS/3’UTR (no ncRNA/Intron/Intergenic calls). Here, for each of the 6 cutoff groups, we restrict our sites to those annotated 5’UTR/CDS/3’UTR and report the % that overlap a BACS site — an apples-to-apples comparison with the BACS dataset.

utr_cds_table <- do.call(rbind, lapply(cutoff_combos, function(cc) {
  idx <- which(anno_sites$fdr < cc$fdr &
               anno_sites$rate_diff > cc$diff &
               anno_sites$depth >= cc$depth)
  sub_utr_cds <- anno_sites[idx, ][anno_sites[idx, ]$feature %in% c("5UTR", "CDS", "3UTR"), ]
